Acerca de

El lenguaje CSS sirve para aplicar estilos y diseñosde forma un poco mas comoda, evitando la molestia de tener que estar repitiendo los códigos cada vez que terminamos una linea, ocasionando en mas de una ocasion que el archivo terminara llenandose de una gran cantidad de codigo que se repite una y otra y otra vez llegando a ser tedioso y restando bastante tiempo a la hora de insertar el código, siendo un ejemplo este:

"-h1--fontface="Arial"--fontsize="5"--fontcolor="blue"-Ejemplo-/font--/font--/h1-

Sin usar CSS este comando habria que repetirse cada vez que lo requirieramos en todo el código, pero ahora si se usa CSS podemos simplemente usar la etiqueta "Style" y este aplicara el estilo a todas las etiquetas que hayamos designado su propio estilo, siendo como ejemplo este código realizado con CSS:

"-style- h1{ font-family: Arial; font-size: 1px; color:blue; }"

Con este comando todas las etiquetas -h1- que vayamos a asignar en esta página tendran el estilo que especificamos en la linea de comando; es bueno tener en cuenta que no es una solución perfecta, ya que aun hay que repetir los comandos a cada página que fuera a crearse, pero al menos de esta forma no hay que repetirlas de forma innecesaria en la misma página.

Eso no significa que solo debamos usar CSS, esto sirve para agilizar la elaboracion del diseño de nuestra página, pero aun asi necesitaremos una base en donde trabajarlo, siendo este el HTML.

HTML seria la base estructural de la página mientras que CSS es la base de la presentación.

Estructura del código

Cada instrucción que se realiza esta dividida en dos partes: El Selector, aquel elemento o etiqueta que sera modificado; y la Declaración, la cual siempre ira entre llaves, esta misma tambien se divide en dos partes:

-La Propiedad, el atributo que sera modificado, por ejemplo el color

-El Valor, lo que designa como sera el atributo asignado

La propiedad se debe de separar de el valor mediante dos puntos; este ultimo siempre dependera del atributo asignado, algunos requiriendo solo numeros u otros requiriendo una medida de longitud en especifico, como hay algunos que usan un valor escrito en especifico, como "left" y "right", deben de siempre estar separado o finalizar cada atributo con un punto y coma como por ejemplo:

p{font-size:5px;}

h1{color:#000066; text-align: "center";}

Esto hara que las etiquetas asignadas a -p- tengan cierto tamaño de fuente, mientras que las etiquetas -h1- sean de color azul y se alineen al centro del renglon.

Y eso seria todo lo que habria que saber acerca de la estructura del código CSS.
